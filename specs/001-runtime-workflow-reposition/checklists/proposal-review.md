# Proposal Review Checklist (Layer-Aware)

- [x] Proposal declares exactly one `owned_layer`.
- [x] Scope boundary is explicit and testable.
- [x] API impact is documented.
- [x] Compatibility classification is one of: non_breaking, deprecation_compatible, breaking.
- [x] Breaking changes set `requires_migration_guide: true`.
- [x] Deprecation-compatible changes include `replacement_path`.
- [x] Test plan is non-empty.
- [x] Documentation plan is non-empty.
- [x] Acceptance criteria are measurable.
